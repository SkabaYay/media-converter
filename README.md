# 📁 Media Converter
A Desktop Application Project to convert files into different types. 

## 🤔 Why?
There are too many shady websites online, so I'm going to create my own.

This project is aimed for me to pursue further understanding of python, software development, and this platform.

Also, I'm primarily making this for personal use (editing).

## 🗣️ How it Works
1. Open the program (**very difficult**).
2. There are two entry box. Put the file type of your file in the first box, and the file type you want to get from the conversion in the second box.
3. Options will appear depending on the type of conversion. Select your file/url and your desired directory.
4. Press the convert button and wait (**also very difficult**).
5. You should now get the converted file in your desired directory.
